# Stock Analysis Streamlit App

This is a Streamlit-based stock analysis app.

Run locally:

```bash
pip install -r requirements.txt
streamlit run app.py
```

Deploy: Connect this repository to Streamlit Community Cloud (https://streamlit.io/cloud) and deploy `app.py` from `main` branch.
